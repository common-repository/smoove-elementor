Sends Elementor forms submitted contacts into your smoove account.

== Description ==
smoove plugin will send contacts details via POST methode to smoove's API
smoove is a marketing tool that allows you to create newsletters, SMS, marketing automations, CRM and more
link to the service:
https://rest.smoove.io 
link to the service terms of use and privacy policies:
https://www.smoove.io/privacy-policy.aspx

== Installation ==

1. In your Smoove account create an API key.
2. Please make sure that you have installed Elementor.
3. Go to plugins in your dashboard and select "Add New".
4. Search for the `Smoove connector for Elementor forms`, install and activate.
5. Enter the “Smoove settings” in the plugin’s menu and enter the API key and follow the instructions in the description.
